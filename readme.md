# Recursion Checkpoint - Palindrome

This project tests if a word is a palindrome using recursion in JavaScript.

A palindrome is a word that can be read the same from left to right and right to left.

Examples:
- radar
- kayak
- php

---

# Recursive Solution

The recursive solution:
- compares the first and last characters
- continues checking the remaining word
- stops when the word is empty or has one character

---
